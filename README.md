# css-keyframes-animation

A CSS Keyframe Animations Showcase
A creative exploration of modern CSS animations, featuring smooth motion paths, directional control, and complex keyframe transitions.
This project demonstrates how to manipulate HTML elements using pure CSS3 animations without any JavaScript dependencies.

‚ú® Key Features

 Linear Horizontal Motion: A blue rectangle (box1) that glides smoothly from left to right using linear timing.

 Synchronized Alternation: A second rectangle (box2) that moves in the opposite direction with an ease-in effect.

 Complex Path Animation: A central circle that follows a multi-point coordinate path (0% to 100%) to create a dynamic, "bouncing" visual effect.

 Fluid UI: Uses alternate and alternate-reverse directions to ensure the motion never snaps and feels continuous.
